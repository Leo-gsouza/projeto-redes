Diferença entre NAME e ID

Name é mais util para HTML e CSS e o ID mais util para a liguagem de programação

Label = Tem a função de relacionar o input ao seu elemento. No primeiro caso acima o label Nome está relacionado ao ID inome

Quando usar GET e POST

Usar Get (padrão)

  • Dados simples
  • Se quiser que a url seja compartilhada
  • O formulario não utrapassar 3 mil bytes = 3 mil letras + ou -

Usar Post

  • Dados sensiveis - menos senhas de cartoes entre outras informações muito sensiveis (estudar https)
  • O formulario utrapassar 3 mil bytes = 3 mil letras + ou -
  • Se desejar enviar arquivos
